# 🩺 AI Healthcare Bot

AI-powered healthcare chatbot and disease prediction system built using Python, Flask, Machine Learning, and SQLite.

## 🚀 Features

* Disease Prediction Based on Symptoms
* Interactive Chatbot Interface
* Disease Information Fetching
* Authentication System
* Symptom Checker
* Nearby Hospital Search
* Machine Learning Integration
* Responsive UI

---

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Python
* Flask

### Database

* SQLite

### Machine Learning

* Scikit-learn
* Pandas
* NumPy

---

## ⚙️ Installation

```bash
git clone <repo-link>
cd AI-Healthcare-Bot

pip install -r requirements.txt

python app.py
```

---

## 🌐 Run Locally

Open browser:

```bash
http://127.0.0.1:3000
```
